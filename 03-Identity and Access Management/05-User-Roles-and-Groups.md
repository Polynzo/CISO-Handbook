# User Roles and Groups

## Standard Roles
- Intern
- Employee
- Team Lead
- Manager
- Admin

## Access Summary
| Role      | Access Level    |
|-----------|-----------------|
| Intern    | Read-only       |
| Employee  | Read/Write      |
| Team Lead | Moderate Admin  |
| Manager   | Full Department |
| Admin     | Full System     |

## Review
Role assignments must be audited every 90 days.
